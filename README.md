Web-equistory это демонстрационное приложение. Приложение имитирует систему контроля версий. С левой стороны навигационное дерево имитируещее файловую систему. Если кликнуть на файл то в середине появятся коммиты отсортированные по дате их создания. Выбрав два коммита мы увидим разницу в коде этих коммитов в правом столбце. 

Приложение написано используя 
* [React](https://reactjs.org/)
* [React-Bootsrap](https://react-bootstrap.github.io/) 
* [React-treebeard](https://github.com/storybooks/react-treebeard)
* [Js diff](https://github.com/kpdecker/jsdiff)
* Webpack
